<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold dark:text-white text-gray-900 mb-4">Quick Start</h2>
  <p className="text-lg dark:text-gray-400 text-gray-600 mb-8">Get Faction running in minutes</p>
  
  <Steps>
    <Step title="Install with Docker">
      Clone the repository and start the containers:
      
      ```bash
      git clone https://github.com/factionsecurity/faction.git
      cd faction
      docker-compose up --build
      ```
      
      <Note>MongoDB requires a CPU with AVX support. Check compatibility if using VirtualBox or Kubernetes.</Note>
    </Step>
    
    <Step title="Access Your Instance">
      Navigate to `http://127.0.0.1:8080` in your browser. On first boot, you'll be prompted to create an admin account.
    </Step>
    
    <Step title="Import Vulnerability Templates">
      Go to **Templates** → **Default Vulnerabilities** and click **Update from Faction** to import 75+ pre-populated vulnerability templates.
    </Step>
    
    <Step title="Create Your First Assessment">
      Create a new assessment, assign team members, and start collaborating. Use the [Burp Suite extension](/integrations/burp-suite) to send findings directly from Burp to Faction.
    </Step>
  </Steps>
</div>
